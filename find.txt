Here is the complete find summary:

MORM find Query
Signature
tsModel.find(clause?)
Single object argument. MORM detects what you want based on keys present.

Basic Usage
ts// All rows
morm.user.find()
morm.user.find({})

// With where
morm.user.find({ where: { active: true } })

// Options only — MORM detects, returns all rows with options applied
morm.user.find({ include: { name: true } })

where — Conditions
Basic values:
ts{ where: { name: "moses" } }       // equal
{ where: { age: 18 } }             // equal
{ where: { active: true } }        // equal
{ where: { deleted_at: null } }    // IS NULL
{ where: { deleted_at: "" } }      // = ''
Operators:
ts{ where: { age: { eq: 18 } } }           // equal
{ where: { role: { not: "ADMIN" } } }    // not equal
{ where: { deleted_at: { not: null } } } // IS NOT NULL
{ where: { age: { gt: 18 } } }           // greater than
{ where: { age: { gte: 18 } } }          // greater than or equal
{ where: { age: { lt: 65 } } }           // less than
{ where: { age: { lte: 65 } } }          // less than or equal
{ where: { age: { gte: 18, lte: 65 } } } // range
{ where: { name: { contains: "mos" } } }    // LIKE '%mos%'
{ where: { name: { startsWith: "mo" } } }   // LIKE 'mo%'
{ where: { name: { endsWith: "ses" } } }    // LIKE '%ses'
Array columns:
ts{ where: { tags: { hasAny: ["javascript", "typescript"] } } }  // contains any
{ where: { tags: { hasAll: ["javascript", "typescript"] } } }  // contains all
Logical:
ts// AND — default for multiple keys
{ where: { role: "ADMIN", active: true } }

// AND — explicit
{ where: { and: [{ active: true }, { verified: true }] } }

// OR
{ where: { or: [{ role: "ADMIN" }, { role: "STAFF" }] } }

// Nested
{ where: { active: true, or: [{ role: "ADMIN" }, { role: "STAFF" }] } }

include — Projection
ts// Specific columns only
{ include: { name: true, email: true } }

// All columns — empty include
{ include: {} }

// Relation — all columns
{ include: { name: true, profile: true } }

// Relation — specific columns
{ include: { name: true, profile: { include: { id: true, avatar: true } } } }

// Relation — with where, orderBy, pagination
{
  include: {
    name: true,
    profile: {
      where: { verified: true },
      include: { id: true, avatar: true },
      exclude: { secret: true },
      orderBy: { created_at: "desc" },
      take: 5,
      page: 1,
    }
  }
}

// Relation — with aggregation
{
  include: {
    name: true,
    orders: { count: true, sum: "amount", where: { status: "completed" } }
  }
}

exclude — Projection
ts// All columns except these
{ exclude: { password: true, secret: true } }

// With relation
{
  include: {
    name: true,
    profile: {
      exclude: { secret: true }
    }
  }
}
Preference order: include > exclude > all columns

orderBy — Sorting
ts// Single column
{ orderBy: { name: "asc" } }

// Multiple columns
{ orderBy: { name: "asc", created_at: "desc" } }

take and page — Pagination
ts{ take: 10, page: 2 }   // 10 rows, page 2

after — Cursor Pagination
ts{ take: 10, after: "uuid-of-last-row" }
// Cannot be used with page

distinct — Unique Rows
ts{ distinct: { role: true, department: true } }

Aggregations
When any aggregation is present → returns single object instead of array:
ts{ count: true }
{ sum: "price" }
{ avg: "score" }
{ min: "age" }
{ max: "age" }

// Combined
{ count: true, sum: "price", avg: "score", min: "age", max: "age" }
// { count: 150, sum: { price: 45000 }, avg: { score: 72.5 }, min: { age: 18 }, max: { age: 65 } }

Relations with Aggregation
tsmorm.user.find({
  include: {
    name: true,
    orders: {
      count: true,
      sum: "amount",
      where: { status: "completed" }
    }
  }
})
// [
//   { name: "Moses", orders: { count: 12, sum: { amount: 3500 } } },
//   { name: "Jane",  orders: { count: 5,  sum: { amount: 1200 } } },
// ]

Return Values
ts// Normal — always array
[]
[{ id: 1, name: "Moses" }]

// Aggregation — single object
{ count: 150, sum: { price: 45000 } }

// With nested relation aggregation — array of objects
[
  { name: "Moses", orders: { count: 12, sum: { amount: 3500 } } },
]

Case Insensitive
All keys work in any case:
ts{ WHERE: { name: "moses" } }
{ Include: { name: true } }
{ ORDERBY: { name: "asc" } }
{ TAKE: 10, PAGE: 2 }
